Latest Project

Fire Engine

Together with @Schone1 I've developed a multi-user-based turntable ladder simulation for virtual reality. The goal of this project was to allow a firefighter squad to practice an incendiary in virtual reality. The playable scenarios are not only inexpensive in comparison to a real scenario setup, but also free from health risks.

Technologies used include:

  • Unity
  • LeapMotion
  • ForceSeatPM
